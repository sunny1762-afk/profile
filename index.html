SKILLS

디자인부터 개발까지 능숙한 툴 활용을 바탕으로 아이디어를 실제 인터페이스로 구현하며, 완성도 높은 결과물을 만들어가는 디자이너입니다.

  • photoshop

    이미지 보정, 색감 조정, 합성 등 다양한 시각 콘텐츠 제작을 할 수 있습니다. 배너, 상세페이지 등 실제 서비스에 적용되는 비주얼 작업을 목적에 맞게 제작할 수 있습니다.

  • Illustrator

    아이콘, 그래픽 요소, 벡터 기반 디자인을 제작할 수 있습니다. 형태와 균형을 고려한 시각 요소를 설계하여 브랜드 이미지에 맞는 그래픽을 구성할 수 있습니다.

  • Figma

    사용자 흐름을 고려한 UI 설계와 프로토타입 제작을 할 수 있습니다. 컴포넌트와 오토레이아웃을 활용하여 일관된 디자인 시스템을 구축하고 인터랙션을 설계할 수 있습니다.

  • Html5

    시맨틱한 구조를 고려한 마크업을 설계할 수 있습니다. 콘텐츠의 의미와 구조를 명확하게 전달하여 유지보수와 확장성을 고려한 코드를 작성할 수 있습니다.

  • Css3

    반응형 레이아웃과 다양한 스타일을 구현할 수 있습니다. Flex, Grid를 기반으로 한 구조 설계와 애니메이션, 인터랙션 요소를 통해 사용자 경험을 향상시킬 수 있습니다.

  • JavaScript

    사용자 인터랙션과 동적인 기능을 구현할 수 있습니다. 이벤트 처리와 DOM 조작을 통해 실제 사용자 행동에 반응하는 UI를 구현할 수 있습니다.

  • Git

    프로젝트의 변경 이력을 체계적으로 관리할 수 있습니다. 브랜치와 커밋을 통해 작업 흐름을 정리하고 안정적인 개발 환경을 유지할 수 있습니다.

  • GitHub

    코드 저장소를 관리하고 협업을 진행할 수 있습니다. 원격 저장소를 기반으로 작업을 공유하고 협업 과정에서 효율적인 커뮤니케이션을 할 수 있습니다.

  • Notion

    프로젝트를 체계적으로 정리하고 협업을 관리할 수 있습니다. 작업 흐름과 내용을 구조적으로 정리하여 효율적인 커뮤니케이션을 할 수 있습니다.
